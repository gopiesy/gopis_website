
1) Make sure node is installed on your PC, if not follow the below link
	https://nodejs.org/en/download/package-manager/
	
2) Make sure express is installed "npm install express --save"

3) From the command line run the command "node install"

4) To run the node server execute the command "DEBUG=gopi-web-portal:* npm start"

5) Access the website on any browser window using url "http://localhost:3000/index.html"
